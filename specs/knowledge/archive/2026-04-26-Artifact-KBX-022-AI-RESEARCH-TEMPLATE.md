## 🏷️ 4. ARTIFACT TAGGING (AI Parse-able)
To ensure artifacts are "logged and tagged" for automatic ingestion:

1. **Metadata Files:** Create a `.json` file in `artifacts/` named after the artifact.
2. **Standard Fields:** Use `artifact_name`, `type`, `version`, and `tags`.
3. **Traceability:** Link the artifact to a specific research log or journal entry.

Example `artifacts/my-artifact.json`:
```json
{
  "artifact_name": "Kernel-Hardening-Audit",
  "type": "Security Audit Report",
  "version": "1.0.0",
  "tags": ["security", "kernel", "hardening"],
  "related_log": "specs/knowledge/research/01-security-audit.md"
}
```
