Enhanced Session · Full Desktop via xrdp Phase 1 · Alternate Launch

Alternate launch path that gives you a FULL GNOME desktop inside the dev VM, RDP'd to Windows via mstsc.exe (the same Hyper-V Enhanced Session client). Complements the per-window WSLg path: MiOS Apps shortcuts keep working, plus a new MiOS Full Desktop shortcut for the unified GNOME session. Packages · what gets layered into the image Phase 7-8 · Install → Build
packages.<section>.enable

Each toggle controls whether [packages.<name>].pkgs is installed during the OCI build. Default = everything ON (full MiOS: host, hosting, dev, build, run, deploy, gaming/SteamOS, AI, Kubernetes, virt, GPU compute) except bloat (gnome-tour, PackageKit, malcontent — opt-in only). Hover for description.

Deployment targets · output image formats Phase 9 · Deploy
deployment.target_<format>

After the OCI image is built, bootc-image-builder renders one artifact per enabled format under /var/lib/mios/build/output/. Default = vhdx + qcow2 + iso + raw + wsl ON (every portable target). AMI / OVA / Anaconda installer are OFF by default — they need extra host tools (aws-cli, ovftool, anaconda).

Services · per-Quadlet first-boot enablement Phase 9-10 · Deploy → Boot
quadlets.enable

Each toggle controls whether the corresponding /etc/containers/systemd/<name>.container Quadlet is enabled at first boot. Disabled Quadlets stay on the image (so re-enabling later is one toggle + reboot) but their generated .service is masked.

Windows bootstrap · M:\ partition + dev VM irm|iex Pass-2

Get-MiOS.ps1's Initialize-DataDisk shrinks C:\ and creates the MiOS data partition. Default shrink_mb = 262656 (256 GiB + 512 MB buffer for NTFS reserved space) -- the bare 262144 MB shrink shows up as "255 GB" in Explorer because NTFS reserves ~16 MB at the start.

bootstrap.host_storage
bootstrap.dev_vm — maximalist allocation
bootstrap.dev_vm.host_reserve — Windows headroom

Applied only when cpus / memory_mb / disk_size_gb is "max". Tuned so Explorer / browser / IDE stay responsive during builds.
