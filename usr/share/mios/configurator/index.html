Appearance · color palette Phase 4 · Overlay
colors (palette applied across every console / terminal / tty / oh-my-posh)

Defaults are the MiOS unified palette: operator neutrals (#1A407F deep-blue, #E0E0E0 silver, #B7C9D7 pale blue-grey, #948E8E warm grey, #734F39 brown) combined with Hokusai's "The Great Wave off Kanagawa" (#3E7765 wave-green, #E7DFD3 foam-cream, #F35C15 sunset-orange, #DC271B coral-red, #282262 deep-indigo). Edit any color below; on save, /etc/profile.d/ mios-colors.sh reads these values and emits OSC-4 / OSC-10 / OSC-11 / OSC-12 escape sequences at every interactive shell start so bash / zsh / PowerShell / Windows Terminal / Ptyxis all repaint. The oh-my-posh theme + Cockpit dark theme + the configurator HTML itself read the same SSOT.

Services · per-Quadlet first-boot enablement Phase 9-10 · Deploy → Boot
quadlets.enable

Each toggle controls whether the corresponding /etc/containers/systemd/<name>.container Quadlet is enabled at first boot. Disabled Quadlets stay on the image (so re-enabling later is one toggle + reboot) but their generated .service is masked.
