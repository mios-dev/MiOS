MiOS Hermes Dashboard - backend-only mode

The React SPA isn't installed in this image. The dashboard service is still up and serving its JSON API under /api/. Use the OWUI chat surface or curl the API directly.

Session-token middleware protects /api/*. The token is injected as window.__HERMES_SESSION_TOKEN__ below; copy it and send as a Bearer header.

Kanban API quick examples

TOKEN=$(curl -s http://localhost:9119/ | grep -oP '__HERMES_SESSION_TOKEN__="\K[^"]+')
curl -s http://localhost:9119/api/plugins/kanban/boards \
  -H "authorization: Bearer $TOKEN" | jq
curl -s http://localhost:9119/api/plugins/kanban/tasks \
  -H "authorization: Bearer $TOKEN" | jq

To install the SPA

cd /tmp && git clone --depth 1 https://github.com/NousResearch/hermes-agent.git
cd hermes-agent/web && npm install && npm run build
sudo cp -r ../hermes_cli/web_dist/* /usr/share/mios/hermes-agent/web_dist_stub/
sudo systemctl restart hermes-dashboard

CLI access is always available regardless: hermes kanban list / hermes kanban create --title ....
