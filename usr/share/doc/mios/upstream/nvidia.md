# NVIDIA on MiOS

> MiOS uses `:stable-nvidia` from ucore-hci (proprietary driver,
> akmod-built, MOK-signed). NVIDIA Container Toolkit + CDI is the path
> for containerized GPU workloads. Source: `usr/share/doc/mios/concepts/architecture.md` §Pillars,
> `usr/share/doc/mios/40-kargs.md` §VFIO, `automation/34-gpu-detect.sh`.

## Stack

| Layer | Component | Where |
| --- | --- | --- |
| Driver kernel modules | akmod-built, MOK-signed at base-image build time | ucore-hci `:stable-nvidia` |
| User-space libs | `libnvidia-*`, `libcuda` | Installed by ucore-hci |
| Container runtime hook | NVIDIA Container Toolkit | Installed by ucore-hci |
| Device interface | CDI (`/var/run/cdi/`) | Generated by `nvidia-ctk cdi generate`; admin overrides `/etc/cdi/` |
| Wayland modesetting | `nvidia-drm.modeset=1` karg | `usr/lib/bootc/kargs.d/00-mios.toml` |
| Memory init disabled | `init_on_alloc=1`/`init_on_free=1`/`page_alloc.shuffle=1` NOT set | CUDA incompatibility — `SECURITY.md` §Kargs |

## akmod & MOK signing

`akmod-nvidia` builds the kernel module against the running kernel and
signs it with the MOK (Machine Owner Key) baked into the base image.
Because MiOS is locked-down (`lockdown=integrity`), only signed
modules load. MiOS doesn't re-sign — the base image's MOK is trusted by
the bootloader chain.

## NVIDIA-LTS variant

For older GPUs (Maxwell / Pascal) needing the proprietary 580 LTS
driver, override `MIOS_BASE_IMAGE`:

```bash
MIOS_BASE_IMAGE=ghcr.io/ublue-os/ucore-hci:stable-nvidia-lts just build
```

Note: `:stable-nvidia-lts` is an **ucore-hci tag**, not a separate MiOS
image tag (the v1 KB had this wrong).

## Container GPU access

```bash
# Generate CDI spec for the host's NVIDIA devices
sudo nvidia-ctk cdi generate --output=/var/run/cdi/nvidia.yaml

# Run a CUDA workload via Podman (CDI device ID format)
podman run --rm --device nvidia.com/gpu=all \
  docker.io/nvidia/cuda:12.4.0-base-ubi9 nvidia-smi
```

The CDI spec lists every GPU by UUID; admins can pin specific GPUs to
specific containers via `/etc/cdi/<override>.yaml`.

- NVIDIA Container Toolkit: <https://github.com/NVIDIA/nvidia-container-toolkit>
- CDI spec: <https://github.com/cncf-tags/container-device-interface>

## VFIO passthrough (MiOS specifically)

For passthrough to a guest VM, see `usr/share/doc/mios/40-kargs.md`
§VFIO. The host detects passthrough-eligible devices at runtime via
`automation/34-gpu-detect.sh` writing
`/run/mios/gpu-passthrough.status`.
